File Finder CLI

Read-only Microsoft 365 file discovery for recent documents, targeted OneDrive search, and script-friendly output.

View on GitHub Get Started

Scope

Uses `Files.Read` only. No write access and no SharePoint-wide escalation.

Commands

  • recent for recently touched files
  • search for focused OneDrive lookup
  • --output json for automation

Auth

Built on the shared Windows-first auth foundation with WAM-first sign-in for lightweight onboarding.

Quick Start

$env:FILE_FINDER_CLIENT_ID='e02be6f7-063a-46a6-b2cc-109d5f51055c' $env:FILE_FINDER_TENANT_ID='a3599b15-c39c-4b41-a219-7e24dd5b5190' $env:FILE_FINDER_SCOPES='Files.Read' $env:FILE_FINDER_AUTH_MODE='wam' $env:FILE_FINDER_ALLOW_BROKER='true' .\invoke.ps1 recent --limit 10
